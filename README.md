## Run Locally

1. Clone repo

   ```shell
    $ git clone git@github.com:basir/next-amazona-v2.git
    $ cd next-amazona-v2
   ```

2. Create .env File

   - duplicate .env.example and rename it to .env

3. Install and Run

   ```shell
     npm install
     npm run dev
   ```

4. Seed Data

   - Run this on browser: http://localhost:3000/api/seed
   - It returns admin email and password and 6 sample products

5. Admin Login
